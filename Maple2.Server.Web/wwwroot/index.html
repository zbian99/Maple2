冒险岛2服务大厅

这里整理了当前 Web 端可用的账号与点卡服务。玩家可以在这里完成账号注册、密码找回和点卡充值，管理人员也可以进入点卡后台。

如果你只是来玩游戏，通常先去注册账号或找回密码，再回到客户端登录即可。

账号注册 统一由 Web 页面办理，客户端只保留登录。
密码找回 支持账号配合 QQ 号或手机号重置密码。
点卡系统 提供玩家充值入口，也提供后台卡密管理入口。
玩家服务

注册账号

创建新的游戏账号，并填写 QQ 号与手机号，用于密码找回和后续账号信息维护。

进入注册
玩家服务

找回密码

通过账号加 QQ 号或手机号进行身份校验，校验通过后可以直接重置登录密码。

找回密码
玩家服务

点卡充值

输入账号、密码和卡密完成充值。充值成功后，时长会自动续到当前账号的到期时间上。

前往充值
管理服务

点卡后台

管理员可在这里登录后台、生成卡密，并查看卡密使用状态和对应账号记录。

进入后台

使用说明

  1. 新玩家先进入“注册账号”，填写账号、密码、QQ 号和手机号。
  2. 忘记密码时使用“找回密码”，至少提供 QQ 号或手机号其中一项。
  3. 已有卡密时进入“点卡充值”，充值成功后即可继续游戏。

快捷入口

  • 服务首页：/
  • 注册页面：/register.html
  • 密码找回：/reset-password.html
  • 点卡充值：/recharge.html
  • 健康检查：/health
健康检查仍可通过 /health 访问，业务接口位于 /api 路径下。
